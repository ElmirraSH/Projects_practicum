### Исследование стартапов

### Цель проекта
Исследовать исторические данные о покупке и продаже стартапов для финансовой компании, которая хочет войти на инвестиционный рынок.
---

### Этапы работы

1. Знакомство с данными и предобработка
2. Предварительное исследование
   - 2.1. Раунды финансирования по годам
   - 2.2. Люди и их образование
   - 2.3. Можно ли использовать столбцы с именем network_username для объединения данных
   - 2.4. Проблемный датасет и причина возникновения пропусков
3. Исследовательский анализ объединённых таблиц
4. Итоговый вывод и рекомендации

### Основные выводы

- К 2013 году количество раундов финансирования растёт, но средний объём инвестиций на один раунд снижается — рынок становится более активным и доступным для стартапов.
- Типичный объём финансирования находится в диапазоне 0–5,63 млн, медианное значение — 0,6 млн; суммы свыше 14 млн считаются выбросами.
- Среди компаний, купленных за $0–1, объём финансирования минимален; верхняя граница выбросов достигает ≈35,8 млн.
- Топ-5 категорий стартапов с наибольшими ценами и разбросом оценок: automotive, sports, nanotech, real_estate, manufacturing.
- Стартапы, вышедшие на IPO, в среднем проходят больше раундов финансирования (медианное значение 2), чем компании с иными статусами (медианное значение 1).

### Рекомендации

- Сфокусироваться на множестве небольших инвестиций.Тенденция снижения объём средств на один раунд и роста количества сделок, стратегия делать много инвестиций с небольшими чеками и диверсифицировать портфель выглядит перспективной.
- Создать модель для оценки перспективности стартапов
- Выбросы до 5,7 млрд среди провальных сделок говорят о переоценке некоторых стартапов в прошлом.
